Terminal:
Ctrl-A -> курсор в начало строки
Ctrl-E -> курсов в конец строки
Ctrl-D -> удаление текущего символа, без перемещения курсора

VSCODE:
Ctrl-L -> выделить строку
Ctrl-R -> поменять среду разработки
Ctrl-Tab -> переключиться между файлами
Ctrl-F2 -> выделить все одинаковые слова
Shift-Del -> удалить всю строку		
F2 -> переименовать переменную
F12 -> перейти к выделенному классу или функции
Сtrl-Shift-A -> длинный комментарий

BROWSER
Ctrl-W -> закрыть вкладку
Ctrl-N -> создать новую вкладку
